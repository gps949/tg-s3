# Telegram Bot コマンドリファレンス

[English](bot-commands.md) | [中文](bot-commands.zh.md) | [日本語](bot-commands.ja.md) | [Français](bot-commands.fr.md)

## 概要

TG-S3 Bot は S3 ストレージを Telegram から管理するためのインターフェースを提供します。すべてのコマンドは指定されたストレージグループまたは Bot との DM で使用できます。

## コマンド

### /start

ウェルカムメッセージを表示し、簡単な紹介とクイックスタートガイドを案内します。

### /help

構文と使用例を含む完全なコマンドリファレンスを表示します。

### /buckets

すべてのバケットをオブジェクト数と合計サイズとともに一覧表示します。

```
/buckets
```

出力例：
```
Buckets (3):
  default - 42 objects, 156.3 MB
  photos  - 128 objects, 1.2 GB
  backup  - 7 objects, 89.5 MB
```

### /ls

バケット内のオブジェクトを一覧表示します。プレフィックスによるフィルタリングも可能です。

```
/ls <bucket> [prefix]
```

- バケット指定：指定バケットのオブジェクトを一覧表示
- プレフィックス指定：キープレフィックスでフィルタリング（ディレクトリ一覧のように機能）

使用例：
```
/ls photos
/ls photos 2024/january/
```

### /info

特定のオブジェクトの詳細情報を表示します。

```
/info <bucket> <key>
```

出力内容：サイズ、Content-Type、ETag、アップロード日時、バケット名。

### /search

指定バケット内でキーパターンでオブジェクトを検索します。

```
/search <bucket> <query>
```

クエリは指定バケット内のオブジェクトキーに対する部分文字列検索で照合されます。

### /share

ファイルの共有リンクを作成します。オプションで制限を設定できます。

```
/share <bucket> <key>
```

オプションパラメータをキーの後に追加できます：

```
/share <bucket> <key> [有効期限_秒] [パスワード] [最大ダウンロード数]
```

- **有効期限**：有効期間（秒）（デフォルト：無期限）
- **パスワード**：パスワード保護（デフォルト：なし）
- **ダウンロード上限**：ダウンロード回数制限（デフォルト：無制限）

生成されるリンクの形式：`https://your-worker.workers.dev/share/<token>`

共有リンクは以下のパスに対応しています：
- `/share/<token>` -- メタデータ付きプレビューページ
- `/share/<token>/download` -- ダイレクトダウンロード
- `/share/<token>/inline` -- インライン表示（画像、動画）

### /shares

アクティブな（期限切れでなく、回数上限に達していない）共有トークンをすべて一覧表示します。

```
/shares [bucket]
```

- バケット未指定：全バケットの共有を一覧表示
- バケット指定：そのバケットの共有のみ表示

トークン、対象ファイル、作成日、有効期限、ダウンロード回数、パスワード設定状況が表示されます。

### /revoke

アクティブな共有トークンを取り消し、リンクを即座に無効化します。

```
/revoke <token>
```

### /delete

ストレージからオブジェクトを削除します。インラインボタンによる確認が必要です。

```
/delete <bucket> <key>
```

削除はカスケード処理で実行されます。Telegram メッセージ、派生オブジェクト（サムネイル、トランスコード版）、関連する共有トークン、キャッシュエントリがすべて削除されます。

### /stats

全バケットのストレージ統計を表示します。

```
/stats
```

出力内容：総オブジェクト数、総サイズ、バケット数。

### /setbucket

Bot にファイルを直接送信する際のアップロード先デフォルトバケットを設定します。

```
/setbucket <name>
```

### /miniapp

グラフィカル UI を備えた Telegram Mini App をインラインで開き、ファイル管理の全機能を利用できます。

```
/miniapp
```

## ファイルアップロード

Bot に直接ファイル（ドキュメント、写真、動画、音声）を送信するとアップロードされます。ファイルはデフォルトバケットに元のファイル名をキーとして保存されます。

圧縮画像として送信された写真（ドキュメントとしてではなく）の場合、Bot は利用可能な最高解像度のバージョンを保存します。

## コールバックアクション

一部のコマンドはインタラクティブなワークフロー用にインラインボタンを表示します：

- **削除確認** -- `/delete` 実行後に「はい、削除」/「キャンセル」ボタンを表示
- **取り消し確認** -- `/revoke` 実行後に「確認」/「キャンセル」ボタンを表示
- **ページネーション** -- 一覧が長い場合に「次のページ」/「前のページ」ボタンを表示

コールバックデータの TTL は 5 〜 10 分です。ボタンが反応しなくなった場合は、コマンドを再度実行してください。
